# Movie Tracking System

Bu proje PHP ve MySQL kullanılarak geliştirilmiş basit bir film takip uygulamasıdır.

## Özellikler
- Film ekleme
- Film listeleme
- Film güncelleme
- Veritabanı üzerinden veri yönetimi

## Kullanılan Teknolojiler
- PHP
- MySQL
- HTML / CSS

## Proje Amacı
Kullanıcıların izledikleri filmleri kaydedebileceği ve düzenleyebileceği basit bir sistem geliştirmek.
